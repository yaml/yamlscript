Contribute to YAMLScript

Want to help out with YAMLScript? Great! Here are some ways you can contribute:

  • Use YAMLScript -- The more people that use YAMLScript, the more feedback we get, and the better the language becomes.
  • Report Bugs -- If you find a bug in YAMLScript, please report it.
  • Request Features -- If you have an idea for a feature in YAMLScript, please let us know.
  • Write Documentation -- YAMLScript is a new language and needs lots of good documentation.
  • Write Tests -- YAMLScript has a test suite that can always use more tests.
  • Discuss YAMLScript with Us
    • YAMLScript Matrix Chat
    • YAMLScript on IRC
    • YAMLScript in Clojurians Slack
