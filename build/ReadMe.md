build
=====

This directory supports building special output formats for `ys`:

* `--to=glj` - Convert YS code to Glojure code
* `--to=go` - Convert YS code to Go code
* `--to=graal-bin` - Build a GraalVM native-image binary executable
* `--to=go-bin` - Build a Go binary executable
* `--to=wasm` - Build a WebAssembly binary file
